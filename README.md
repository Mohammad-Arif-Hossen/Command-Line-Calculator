
   ```bash

   ```


2. **Manual Start (All Platforms)**:


### üõ†Ô∏è Available Commands

- `help` - Display available commands and functions
- `history` - Show calculation history
- `clear` - Clear screen
- `exit` or `quit` - Exit calculator
